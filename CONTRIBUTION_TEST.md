# Cloud Security VAPT Framework

## Automated Vulnerability Assessment & Penetration Testing

This framework provides automated security testing for:
- AWS (Amazon Web Services)
- Azure (Microsoft Azure)  
- GCP (Google Cloud Platform)

## Features
- IAM Privilege Escalation Discovery
- Multi-Cloud Misconfiguration Scanning
- Infrastructure as Code Security Analysis
- Container Security Testing
- API Security Assessment
